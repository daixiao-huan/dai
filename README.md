# dai

Python数据分析在线教育平台

## 项目简介

这是一个基于React + TypeScript + Vite开发的Python数据分析在线教育平台，专为商务数据分析与应用专业的学生设计。

## 主要功能

- 完整的课程体系（入门、中级、高级）
- 互动式学习模块
- 包含学、练习、测评功能
- 成就激励系统
- 在线代码编辑器（支持Python代码执行）

## 技术栈

- React 18 + TypeScript
- Vite 6
- Tailwind CSS
- React Router
- CodeMirror 6（代码编辑器）

## 部署

项目已部署到Cloudflare Pages。
