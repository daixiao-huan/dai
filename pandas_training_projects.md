# AI时代的数据分析课：10个Python (pandas) 训练项目

## 项目1：基础数据清洗 - 订单数据预处理

### 输入数据格式
```
orders.csv
- order_id: 订单ID (字符串)
- customer_id: 客户ID (字符串)
- order_date: 订单日期 (字符串，格式不统一)
- amount: 订单金额 (数值，可能包含异常值)
- status: 订单状态 (字符串)
- product_id: 产品ID (字符串)
- quantity: 数量 (数值，可能有缺失值)
```

### 任务目标
1. 处理日期格式不统一问题
2. 识别并处理缺失值
3. 检测并处理异常值
4. 数据类型转换

### 输出结论
- 清洗后的订单数据，包含标准化的日期格式、合理的数值范围
- 数据质量报告，包括缺失值和异常值的处理情况

## 项目2：用户行为分析 - 浏览与购买转化

### 输入数据格式
```
user_behavior.csv
- user_id: 用户ID (字符串)
- timestamp: 行为时间戳 (字符串)
- behavior_type: 行为类型 (浏览/加购/购买，字符串)
- product_id: 产品ID (字符串)
- category_id: 品类ID (字符串)
```

### 任务目标
1. 计算不同行为类型的分布
2. 分析用户从浏览到购买的转化路径
3. 识别高转化品类

### 输出结论
- 用户行为漏斗分析报告
- 各品类转化率排名
- 转化时间分布分析

## 项目3：购物篮关联分析 - 商品组合推荐

### 输入数据格式
```
order_items.csv
- order_id: 订单ID (字符串)
- product_id: 产品ID (字符串)
- product_name: 产品名称 (字符串)
- quantity: 数量 (数值)
- price: 单价 (数值)
```

### 任务目标
1. 使用关联规则算法分析商品组合
2. 计算支持度、置信度和提升度
3. 识别强关联商品组合

### 输出结论
- 商品关联规则表，包含支持度、置信度和提升度
- 前10个强关联商品组合推荐
- 关联分析可视化图表

## 项目10：库存预警系统 - 供应链管理

### 输入数据格式
```
inventory_data.csv
- product_id: 产品ID (字符串)
- product_name: 产品名称 (字符串)
- current_stock: 当前库存 (数值)
- safety_stock: 安全库存 (数值)
- average_daily_sales: 日均销量 (数值)
- last_restock_date: 上次补货日期 (日期)
```

### 任务目标
1. 计算库存周转天数
2. 识别库存不足的产品
3. 生成补货建议

### 输出结论
- 库存状态分析报告
- 库存预警产品清单
- 补货计划建议

## 学习成果

完成这10个项目后，学生将掌握：

1. **数据清洗技能**：处理缺失值、异常值、格式统一
2. **数据分析技能**：分组、聚合、透视表、关联分析
3. **业务解读能力**：从数据中提取有价值的业务 insights
4. **可视化能力**：使用pandas和相关库创建数据可视化
5. **实战经验**：解决真实电商/零售场景中的数据分析问题

每个项目都设计为可独立完成，同时难度逐渐递进，确保学生能够循序渐进地掌握pandas的核心功能和数据分析的实战技巧。
